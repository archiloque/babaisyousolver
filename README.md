# Baba Is You solver

A WIP solver for the game [Baba Is You](https://hempuli.com/baba/).

## Usage

Build:

```
mvn install
```

Run all levels:
```
java -jar target/babaisyousolver-1.0-SNAPSHOT.jar 
```
